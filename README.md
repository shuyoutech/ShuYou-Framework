# ShuYou-Framework

ShuYou-Framework微服务脚手架，采用SpringBoot3.5、SpringCloud2025、SpringCloudAlibaba2023等框架，开发的一套SpringCloud开发平台，旨在为开发者提供一个简洁、高效、可扩展的代码开发平台

技术栈

| 名称                  | 版本号        | 备注 |
|---------------------|------------|----|
| java                | 21         |    |
| SpringBoot          | 3.5.4      |    |
| SpringCloud         | 2025.0.0   |    |
| SpringCloud Alibaba | 2023.0.3.3 |    |
|                     |            |    |
